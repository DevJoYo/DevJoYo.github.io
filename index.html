PROJECTS

Simple Blackjack JavaFX application

Project: Almost BlackJack!

This simple JavaFx program was the result of a canceled assignment in my second semester. Due to it being a summer semester the proffesors were swapped six weeks in, however I had already started making this and was enjoying the challenge, so I decided to finish it in my free time.

This program allows you to simulate playing blackjack against a dealer. The cards are dealt at random and the cards that are left in the deck are tracked in a side panel (it counts cards for you, so you should win...). The program also keeps track of how many hands you have won so you can try for a high score. Every time the dealer wins you lose a life, and the game ends when you run out of lives.

Technologies used:

Java, JavaFX

ASP.NET application designed to help veterinary technicians do complex conversions for veterinary medicine

Project: VetTech Calc (Pretend Project)

This is a pretend project that I have not yet worked on but I have been thinking of using the skills I've learned at mohawk to make this application for my partner to use at her job.

This application would allow a user to select the type of animal they were working with, enter information about the animal (weight, breed, etc) and select which drug they are administering, and provide the correct dosage. This would be incredible helpful as veterinary technicians often have to do complex conversions between volume and weight as each drug has strict dosage requirements. I believe a web application would be the most benifical way to ship this application, as the technicians could load it on their phones or tablets as long as they had an internet connection.
